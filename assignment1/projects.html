Summaraize – AI Note Cleaner

This the main page of the website

This is the page where unreadable board photos are uploaded and converted into clean PDFs using AI.

We were announced as the winner

Summaraize is an AI-powered tool developed during the EPAM Generative AI Hackathon, where our team ranked 1st among İzmir universities. The project focuses on solving real student-life problems using generative AI.

The system converts hard-to-read handwritten or board lecture notes into clean and readable PDF documents. The main goal is to help students digitize messy notes, improve readability, and enhance study efficiency.

Technologies: HTML, JavaScript, AI APIs
